# Remembry — Pitch Brief for Friend

---

## Remembry — One-liner
> "Not another meeting transcriber — a privacy-first organizational memory platform for SMEs."

---

## What It Is
- Privacy-first AI meeting platform for SMEs
- Desktop app: upload/record meeting → Gemini AI extracts structured memory → searchable Q&A across all history
- All data stays local on device — nothing goes to cloud
- Users bring their own Gemini API key — no vendor lock-in, no subscription

---

## The Problem (4 Pain Points)
1. **Knowledge lost** — meetings not documented, decisions disappear, recordings never revisited
2. **Employee turnover** — institutional knowledge vanishes when people leave
3. **Cloud privacy risk** — sensitive discussions uploaded to third-party servers
4. **Subscription chaos** — separate tools for transcription, notes, storage = no unified memory

---

## Our Solution (3 Core Features)
1. **Local-first privacy** — all data stays on device, user controls everything
2. **AI-powered extraction** — structured summaries, action items, decisions, Q&A
3. **Searchable memory** — ask "Why was Project Alpha postponed?" get contextual answers

---

## Key Differentiation
- Competitors (Otter, Fireflies, CAST) = transcription tools
- Remembry = **organizational memory** platform
- "Memory vs notes" — we connect decisions across time, not just record them

---

## Why Now (4 Reasons)

**1. AI transcription is cheap & mature**
- Gemini, GPT-4o, Whisper are accurate and affordable
- 2 years ago needed expensive custom ML — now accessible to anyone
- SMEs can adopt without enterprise budgets

**2. Privacy demand is rising**
- GDPR, HIPAA, data residency = compliance is critical
- Uploading sensitive discussions = real risk for SMEs
- Companies want data ownership, not vendor-controlled cloud

**3. Knowledge loss is accelerating**
- Remote work + fast turnover = institutional memory disappears fast
- SMEs most vulnerable — small teams, no knowledge managers
- Meeting overload + nothing preserved = problem getting worse

**4. Market gap is wide open**
- Every competitor focuses on transcription
- Nobody focuses on long-term organizational memory
- Gap between "meeting notes" vs "business intelligence" is huge and unclaimed

---

## Simple Analogy for Friend
> "It's like the difference between writing notes vs actually having a memory. Everyone builds note-taking tools. We're building the memory layer that connects everything over time."

---

## Bottom Line
**Window is open. AI is ready. Privacy problem is real. Market is underserved.**
We are building in a space competitors have largely ignored — right now is the perfect time.
